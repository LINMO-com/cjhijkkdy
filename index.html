安全分享平台

企业级文档统一存储、元数据治理与限时安全分享。基于 HMAC-SHA256 签名的动态下载策略，普通文件 15 分钟、大文件 30 分钟有效，过期即失效。

PUBLIC WEB
网页前端
面向访客的公开文档浏览界面，支持卡片墙展示、格式图标、在线预览与限时下载。
进入前端
  • 文档卡片墙与格式筛选
  • TXT / Markdown / JSON 在线预览
  • 图片灯箱预览
  • 分享链接限时下载
ADMIN DASHBOARD
管理后台
管理员专属控制台，支持文件上传、元数据编辑、重命名、生成分享链接等完整管理功能。
进入后台
  • 拖拽多文件上传
  • 元数据在线编辑
  • 同名冲突检测重命名
  • 一键生成安全分享链接

快速启动 / Quick Start

1
初始化数据库
mysql -u root -p < database/schema.sql
2
启动后端服务 (端口 3000)
cd server && cp .env.example .env && npm install && npm run dev
3
启动前端服务 (端口 5173)
cd web && npm install && npm run dev
4
点击上方卡片进入系统
默认账号: admin / admin123
